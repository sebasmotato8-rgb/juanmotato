## Tips para usar en aplicaciones

- En tu CV en PDF o LinkedIn, incluye el link al repositorio (`github.com/sebasmotato8-rgb/juanmotato`) como "Portafolio / CV en línea".
- Activa GitHub Pages (ver `GITHUB_SETUP.md`) para tener una URL limpia (`.github.io`) que se vea mejor en una tarjeta de presentación o firma de email.
- Actualiza el README cada vez que termines un proyecto nuevo — es lo primero que ve un reclutador al entrar al repo.
